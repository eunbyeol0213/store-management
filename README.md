## 3. 프로젝트 주제
센터 관리 시스템을 통해 입력된 각종 자료는 약점뿐 아니라 강점 파악, 재고 및 담당자 관리 등 관리자에게 필요한 다양한 정보를 제공함으로써 효율성을 향상시켜 줍니다.
또한 권한에 따른 메뉴 접근을 제공하여 보다 체계적인 센터 운영을 제공합니다.
</br>
